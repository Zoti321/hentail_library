# Triage Labels

**This repo does not use the triage label system.**

Skills such as `triage` must not create, apply, or rely on the five canonical triage role labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`).

Manage issue state through titles, bodies, comments, and open/closed status only — not through triage labels.
